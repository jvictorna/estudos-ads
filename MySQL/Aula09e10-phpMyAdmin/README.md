\# 🥗 Prática Complementar — phpMyAdmin, Workbench e MariaDB Console



Prática complementar desenvolvida durante os estudos de MySQL com foco em modelagem,

manipulação e organização de dados utilizando diferentes ferramentas de administração e execução SQL.



\## Ferramentas utilizadas

\- phpMyAdmin

\- MySQL Workbench

\- MariaDB Console



\## Objetivos da prática

\- Criar e estruturar tabelas utilizando SQL

\- Praticar manipulação de dados

\- Utilizar múltiplos ambientes de administração MySQL

\- Exercitar comandos SQL no console MariaDB

\- Compreender erros comuns de sintaxe e manipulação



\## Conceitos praticados

\- CREATE DATABASE

\- CREATE TABLE

\- INSERT INTO

\- SELECT

\- UPDATE

\- ALTER TABLE

\- PRIMARY KEY

\- AUTO\_INCREMENT

\- UTF8MB4

\- manipulação de dados

\- modelagem básica de banco de dados

\- debugging e correção de comandos SQL

\## Observações



Durante a prática, também foram realizados testes no MariaDB Console para reforçar o uso manual dos comandos SQL. Nessa etapa foram identificados e corrigidos erros comuns, como:

\- erros de digitação em comandos SQL;

\- ausência de tipos primitivos;

\- problemas de codificação UTF8 no terminal;

\- aspas simples não fechadas;

\- warnings causados por tamanho insuficiente de colunas VARCHAR.
