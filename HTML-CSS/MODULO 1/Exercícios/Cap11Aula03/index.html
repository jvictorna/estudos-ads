Elemento de Áudio

Reproduzindo Áudio

Vamos aprender a reproduzir áudios em HTML5

Existem duas maneiras de adicionar o reprodutor de áudio dentro do site. A mais simples é usando a tag <audio> + scr, junto com os parâmetros controls e autoplay.

Os formatos que são compatíveis com os principais navegadores são: MP3, WAV e OGG.

A outra maneira de adicionar reprodutor de áudio dentro do site é usando os parâmetros source + type entre a tag <audio>.

Adicionando o áudio com a segunda forma, o navegador que o usuário utiliza irá tentar reproduzir o áudio em MP3, caso não seja compatível irá tentar reproduzir no formato OGG, não sendo compatível tentará com WAV, se ainda assim não for compatível irá exibir a mensagem de falta de compatibilidade. OBSERVAÇÃO IMPORTANTE: para que seja eficiente, é necessário hospedar todos os formatos de áudio.

As mesmas regras de direitos autorais e tamanhos de arquivos valem para elementos de áudios, ou seja, apenas utilizar mídias autorizadas e com tamanhos baixos para evitar problemas de direitos autorais e carregamento do site. (arquivos MP3 e OGG tendem a ter tamanhos menores que WAV, com qualidade reduzida).
