ANOTAÇÕES

Diferente da aplicação de estilos CSS inline, no CSS local as configurações de estilos ficam concentradas em um único lugar e as alterações realizadas são aplicadas em todos os locais que usam aquela tag. Para um site de uma única página pode ser vantajoso, mas existe ainda um outro método (criando classes) que pode ser usado em diversas páginas utilizando a mesma configuração de estilo.
