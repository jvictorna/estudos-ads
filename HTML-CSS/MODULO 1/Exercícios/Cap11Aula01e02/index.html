Imagens dinâmicas

Usando a tag <picture> é possível adicionar imagens e configurar com o parâmetro <source:media> para que o HTML5 ajuste a exibição de acordo com o tamanho do navegador ou dispositivo utilizado para saída.

Para isso, é necessário ter as imagens já adaptadas para os diversos tamanhos de tela.

Segundo o professor Guanabara, isso é algo essencial na criação de um site, pois além de pensar no usuário, pode evitar com que esse usuário saia do site.
