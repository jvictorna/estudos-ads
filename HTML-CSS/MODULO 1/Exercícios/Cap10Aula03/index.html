Para criar link para download utilizamos a tag A envelopando o texto que se deseja dedicar o download e referenciamos o arquivo na pasta em que ele se encontra. É preciso utilizar os parâmetros download="nomedoarquivo" e type="tipo/formato" para que o navegador realize o download do arquivo e não apenas o abra no navegador.
