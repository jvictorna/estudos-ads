Estilos CSS Inline

O uso das CSS inline consiste na utilização da estilização de forma pontual nas tags, ou seja, cada tag recebe diretamente o seu estilo. Exemplo, se houver 3 <h1> dentro de uma mesma página e eu queira que eles tenham uma determinada cor, no CSS inline eu precisarei configurar cada h1 com a cor desejada, mesmo que a cor seja igual para todos.
