ANOTAÇÕES

É possível criar um arquivo exclusivo para adicionar os estilos e linkar esse arquivo com as páginas que se deseje colocar esse mesmo estilo. Facilita bastante o uso de estilos que vão ser iguais para diversas páginas. Também é possível criar arquivos com configurações de estilos específicas e juntar com outros links em uma mesma página.

Também é permitido juntar o método de CSS interno com o CSS externos e CSS inline, sendo que o que se sobrepõe é sempre o inline.
