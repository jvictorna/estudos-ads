Para criar um link externo utilizamos a tag A envolvendo o texto que se quer linkar. Dessa forma, ao clicar no link indicado o usuário será redirecionado para a página destino dentro da mesma página web, ou seja, fechando a página principal sendo necessário clicar em voltar para voltar ao site (o que dificilmente pode ocorrer).

Quando o link redirecionar para páginas externas, é recomendável utilizar o parâmetro target="_blank" rel="external" fazendo com que seja aberta uma nova aba ao clicar no link, evitando a sobreposição da página atual.

É possível criar links internos onde direcionam para outras páginas do site, dentro da mesma pasta do index. No caso de links interno, por se tratar de link interno, não é possível usar o rel="external", mas é possível usar o rel="next" para avançar ou rel="prev" para voltar.
