O HTML5 atualmente prioriza o SENTIDO das TAGS e não apenas a FUNÇÃO.

Nesta frase, temos um termo em destaque usando a tag STRONG (semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

E no outro parágrafo, temos outro texto marcado no final.

Quando a formatação de estilo é feita diretamente na TAG, apenas aquela tag receberá as configurações, em caso de novo uso prevalecerá o padrão.

Para uma TAG receber as características de uma formatação, é necessário criar essa formatação no cabeçalho do código.

Caso seja feito uma configuração de estilo para determinada TAG e eu precise alterar em determinada linha, é possível alterando a TAG dentro dessa linha

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Atualmente, a tag BIG passou a ser considerada obsoleta pois não faz sentido a letra ser grande. Caso precise de letras grandes, isso pode ser alterado através do estilo usando CSS.

Já a tag SMALL ainda tem sentido pois em texto ainda pode se fazer necessário o uso de letras pequenas, como por exemplo, as "letras miúdas" de um determinado contrato.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag U (não semântica).
