ANOTAÇÕES

FONT WEIGHT

É usado para determinar o 'peso' de uma fonte, tornando ela mais ou menos 'gordinha'. Pode ser determinada por normal, lighter, bold, bolder, sendo bolder a mais forte (mais que o negrito) ou também por numeração de 100 a 900, sendo 900 mais forte (mais que o bolder).

Para que a weight funcione é necessário que a fonte selecionada tenha essa variação, geralmente as mais populares não tem.

Para usar a font weight usamos o parâmetro font-weight:lighter;

FONT STYLE

É usado para determinar o estilo italico de uma fonte.

Utilizamos o parâmetro font-style: italic; Também é preciso que a fonte utilizada tenha essa variação de estilo.

TEXT DECORATION

Usado para deixar o texto sublinhado através do parâmetro text-decoration: underline;

SHORTHAND FONT

Shorthand são usadas para 'condensar' comandos, ou seja, é possível compactar um comando de 4 linhas em apenas uma, respeitando a ordem dos parâmetros.

No caso do shorthand font a ordem correta é: font-style -> font-weight -> font-size -> font-family

Exemplo de um comando shorthand para o h1: font: italic bolder 3em sans serif;
