ANOTAÇÕES

Para personalizar um link, dentro da tag style criamos parâmetros de personalização, podendo alterar cor, tamanho dentro outras.

Usamos o parâmetro text-decoration: none quando queremos que os links não fiquem com o sublinhado.

Geralmente ao ser clicado um link fica com outra tonalidade (na maioria das vezes, roxo), caso queira uma alteração dessa cor, criamos uma nova aba de personalização com a pseudo classe :visited, ou seja, ficaria por exemplo a:visited { cor desejada };

Também é possível personalizar para quando ter o mouse passado por cima do link ele fique sublinhado, para isso adicionamos uma nova pseudo classe com o hover.

O :active é usado para personalizar o link quando este for clicado, ou seja, quando estiver ativo.

PSEUDO ELEMENTOS

Para usar pseudo elementos utilizamos o :: após o elemento, por exemplo: a::after { }

Pseudo elementos podem agir nas classes, nos elementos, nos IDs, ou seja, diretamente no conteúdo do elemento.

Por exemplo, se quiser que todo link tenha indicação no elemento usamos o a::after { content: '[link]'; }, ou seja, depois do conteúdo de a entrará a palavra link entre colchetes. Também é possível personalizar especificamente essa palavra dentro desse mesmo parâmetro.

SELETORES PERSONALIZADOS

  • # = id
  • . = class
  • : = pseudo-class
  • :: = pseudo-element
  • > = children
