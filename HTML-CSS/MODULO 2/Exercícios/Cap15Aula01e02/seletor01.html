SELETORES PERSONALIZADOS

CSS id

Usamos o CSS id quando queremos diferenciar a personalização de um determinado elemento dentro do html. Por exemplo, dar uma personalização específica para o h1 do título inicial, que será diferente dos demais h1 do código.

O que em HTML for id = em CSS é #

O que em HTML for class= em CSS é .

Para usar o id, usamos id="nomedoid" dentro da tag do elemento que queremos personalizar, e na aba de estilos sinalizamos a personalização com #. Segue exemplo:

<h1 id="principal">

h1#principal { conteúdo da personalização }

Dessa forma, apenas o h1 sinalizado com id principal irá receber a personalização diferente dos demais h1. No caso do id, só é recomendado o uso de 1 por html, ou seja, não é recomendado utilizar novamente a personalização do principal para outro elemento, mesmo que seja outro h1, para isso, utilizamos o CLASS.

CSS class

Usamos o CSS class quando queremos diferenciar a personalização de diversos elementos. Segue basicamente a mesma regra do id.

Para usar o class, usamos class="nomedaclass" dentro da tag do elemento que queremos personalizar, e na aba estilos sinalizamos a personalização com . (um ponto). Segue exemplo:

<h2 class="basico">

.basico { conteúdo da personalização }

Dessa maneira, todos os elementos que receberem a class basico receberão a personalização específica.

Observação importante: evitar dar nome de acordo com a forma, e sim de acordo com a função. No exemplo, usei básico porque a class vai indicar sub titulos básicos.
