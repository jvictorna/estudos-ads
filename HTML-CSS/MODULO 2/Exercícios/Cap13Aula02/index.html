REPRESENTAÇÃO DE CORES

Representação por nomes

Utiliza o nome das cores em inglês, exemplo: red, green, blue, white, black etc. Corretamente utilizado, porém com grande limitação de cores.

Representação por códigos Hexadecimais

Usa representação de cores através de código hexadecimal, sendo: 2 dígitos para cada cor -- -- --. Exemplo do azul: 00 00 ff sendo representado por #0000ff sendo 00 para ausência de RED, 00 ausência de GREEN e ff o máximo de azul. Ao alterar o nível do RED e GREEN influência na tonalidade do azul.

Representação por RGB

Usa o RGB para representação das cores, sendo RED GREEN BLUE representados por números de 0 a 255. A lógica é a mesma do hexadecimal, porém representado apenas pelos números. Sendo o azul em rgb 0, 0, 255 ou seja ausência de red, ausência de green e máximo de azul.

Transparência usando RGBA

A lógica das cores segue a mesma do RGB, porém com a adição do A que significa ALFA e representa a opacidade da cor, variando de 0 a 1, onde 0 é totalmente transparente e 1 é totalmente opaco.
