PSEUDO CLASSES

Uma DIV é como se fosse um espaço vazio, podendo ser apenas uma linha ou algo com espaços pré-determinados através de altura (height) e largura (width).

As pseudo classes são representadas pelo sinal de : e através delas é possível configurar o conteúdo que fica dentro do elemento. Ou seja: no exemplo, foi criado 3 elementos div, configurados visualmente como quadrados cinzas e ao passar o mouse sobre eles, ficam amarelos. O efeito de ficar amarelo foi através da ligação do elemento com a pseudo classe hover que permite alterações do estado do elemento.
