FONTES EXTERNAS

Também é possível baixar fontes de sites externos e adiciona-las ao arquivo, as fontes externas também podem vir de diretamente do cliente, por exemplo, em arquivo ttf ou otf.

Existe o site dafont.com que disponibiliza fontes gratuitas para download.

Ao utilizar fontes externas é importante verificar se essa disponibiliza todos os glifos: se será possível escrever com letras maiúsculas, minúsculas, com e sem acento.

Para adicionar uma fonte externa utilizamos o parâmetro @font-face { font-family: ; src: url() format(); } onde no url será colocado o nome do arquivo e dentro de format iremos colocar o tipo do arquivo. O ideal e indicado é sempre guardar as fontes dentro de uma pasta separada, e ao colocar o nome do arquivo indicar que o mesmo encontra-se dentro da pasta.

Exemplo de como ficaria com a fonte usada:

@font-face { font-family: Love; src: url('fonts/lovelarry.ttf') format('truetype');}

Para arquivos OTF usamos o opentype.

Para arquivo TTF usamos o truetype.

Existe também o tipo embedded-opentype para alguns casos, e o truetype-aat para APPLE ADVANCED TYPE.
