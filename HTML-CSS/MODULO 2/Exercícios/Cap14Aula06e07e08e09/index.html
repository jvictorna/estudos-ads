GOOGLE FONTS

É uma plataforma do google que disponibiliza fontes de forma gratuita, para usar as fontes após a escolha basta importar o código dentro da tag STYLE.

As fontes importadas vão carregar no navegador caso tenha acesso a internet, caso não tenha, segue a regra da próxima fonte, por isso a importância de sempre colocar para que o navegador carregue a fonte genérica em caso de não conseguir carregar a fonte principal.

É indicado utilizar no máximo entre 3 a 4 fontes em um mesmo site, evitando muita diversificação.

FONTS NINJA

O fonts NINJA consiste em uma extensão do google que permite identificar qual fonte está sendo utilizada em determinado local apenas passando o mouse em cima da fonte. A extensão mostrar o nome da fonte e os tamanhos que estão sendo aplicados.

Essa ferramenta pode ser bastante útil para identificação de fontes aleatórias.

Fontes em imagens

Existem algumas ferramentas que permitem extrair a fonte usada em imagens. Algumas delas são:

  • whatfontis.com
  • fontsquirrel.com
  • myfonts.com

Atenção: é preciso ter atenção pois nem sempre os resultados serão o esperado.
