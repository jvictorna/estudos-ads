ANOTAÇÕES

Utilizamos o parâmetro text-align quando determinar o alinhamento de títulos/subtítulos e textos. Com isso, determinamos se queremos centralizado (center), alinhado à esquerda (left) ou a direita (right), e justificado (justify). O text-indent é usado para criar um espaçamento no início do parágrafo, seguindo as normas ortográficas. Segue exemplos:

  • text-align: left;
  • text-align: right;
  • text-align: center;
  • text-align: justify;
  • text-indent: 30px;

Como essa configuração é uma configuração de estilo e não de sentido, deve ser feita com CSS na aba dedicada a estilos.
