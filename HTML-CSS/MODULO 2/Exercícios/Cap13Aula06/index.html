TESTANDO GRADIENTE

Dentro do cabeçalho do código, utilizamos o parâmetro background-image: linear-gradient(); para criar um fundo com degradê. Dentro do parênteses indica-se a direção em que irá iniciar o degradê, seguido das cores (separados por vírgula). top right para que iniciar na direita, to left na esquerda, to top para iniciar de cima (nesse caso, é necessário criar uma configuração global com * {} e indicar que o tamanho da tela será de 100%), to bottom para baixo.

Também é possível indicar a direção do início do degradê utilizando os graus, para isso usa-se, exemplo 45deg, 60deg (onde a numeração indicará o tamanho do grau, e deg sinaliza que é em graus).
