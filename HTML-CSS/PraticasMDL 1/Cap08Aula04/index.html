Citações simples

A tag Q é usada para realizar citações utilizando aspas, de forma semântica, dando sentido a citação.

JavaScript foi criada na Netscape na fase inicial da Web e, tecnicamente, "Javascript" é marca registrada, licenciada pela Sun Microsystem (agora Oracle), usada para descrever a implementação da linguagem pelo Netscape (agora Mozila).

Usando a tag BLOCKQUOTE é possível criar citações que ficam um pouco mais organizadas e formatadas como citação, além de poder criar um parâmetro CITE e adicionar o link de onde a citação foi tirada fazendo com que mecanismos de buscas consigam identificar de onde a citação foi retirada.
