Para criar listas ordenadas utilizamos a tag OL e dentro da tag OL adicionamos linhas a lista com a tag LI. O padrão da OL é lista numérica, mas é possível alterar isso adicionando o parâmetro type dentro da tag OL, podendo optar por: ordem numérica (padrão), letras maiúsculas(A) e minúsculas(a) ou algarismos romanos(I ou i).

Para criar listas que não precisam seguir uma ordem específica utilizamos a tag UL e também criamos linhas usando a tag LI. Também é possível alterar o caractere que irá organizar essa lista, sendo o padrão o disco (bolinha preenchida), mas tem também a circle (bolinha vazia), square (pequeno quadrado).

Exemplo com circle (bolinha vazia)

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Exemplo com square (pequeno quadrado)

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Atualmente, o atributo type na UL é considerado obsoleto no HTML5 — o ideal é usar CSS para essa configuração.
